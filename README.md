# Botão Switch Customizado

Interface minimalista de botão switch/toggle construída apenas com HTML e CSS.

Este projeto é uma demonstração estática feita com **HTML e CSS**, pensada para funcionar diretamente no navegador e poder ser publicada com **GitHub Pages**.

## Funcionalidades

- input checkbox estilizado como switch
- animação suave do thumb
- texto que muda de Disabled para Enabled
- layout centralizado com gradiente

## Estrutura do projeto

- `css/style.css`
- `img/icone.png`
- `index.html`

## Visual

O projeto usa um gradiente azul/verde e um componente branco centralizado.

## Como executar

1. Clone ou baixe este repositório.
2. Abra o arquivo `index.html` diretamente no navegador.
3. Se preferir usar um servidor local, execute na pasta do projeto:

```bash
python -m http.server 8000
```

Depois acesse:

```text
http://localhost:8000
```

## GitHub Pages

Como este projeto é estático, ele pode ser publicado no GitHub Pages usando a branch principal.

Configuração recomendada:

- Branch: `main`
- Pasta: `/ (root)`

## Tecnologias

- HTML5
- CSS3
